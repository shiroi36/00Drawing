# 個別計算書Juliaプログラム

### プログラムの概要
本プログラム群はJuliaで書かれた構造計算プログラムである。
プログラムはおおよそ変数の定義とprint出力が交互に書かれた形式となっており、
各変数の意味は直後のprint出力の文字列で確認できる。
各プログラム先頭のコメントに検討内容の説明を示した。

### 使用方法
1. Juliaを[公式ドキュメント](https://julialang.org/downloads/)に従ってインストールする。
2. プログラムの各変数を適宜修正する。
3. プログラムを実行し、出力を確認する。
```
julia {filename}.jl
```
4. 出力が問題なければクリップボードにコピーする。PowerShellであれば以下を実行する。
```
julia {filename}.jl | clip
```
5. コピーした出力を計算書にペーストする。
