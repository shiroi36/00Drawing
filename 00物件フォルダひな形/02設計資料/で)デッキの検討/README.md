# デッキスラブの検討

### Sデッキの場合
https://www.itec-c.co.jp/products/ より計算プログラムをダウンロードする。
同封のマニュアルに従ってExcelを入力する。
PDF出力した計算書をコピーペーストしてWordに貼り付ける。

### QLデッキの場合
https://decks.jfe-kenzai.co.jp/ のWebアプリで計算する。
利用には会員登録が必要である。
同ページに記載のマニュアルに従ってフォームを入力する。
PDF出力した計算書をコピーペーストしてWordに貼り付ける。
入力したデータは「設計データリスト」からCSVとして入出力可能である。

### MAデッキの場合
https://www.meiji-kohgyo.co.jp/download/download03/ より計算プログラムをダウンロードする。
同封のマニュアルに従ってExcelを入力する。
Excel出力した計算書をコピーペーストしてWordに貼り付ける。

### スーパーEデッキの場合
https://decks.ns-kenzai.co.jp/ のWebアプリで計算する。
利用には会員登録が必要である。
同ページに記載のマニュアルに従ってフォームを入力する。
PDF出力した計算書をコピーペーストしてWordに貼り付ける。
入力したデータは「Revit連携」機能でXMLとして入出力可能である。
